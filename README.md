The FRED Data Pipeline is a simple retrieveing tool that utilizes the fredapi Python wrapper to connect to the Federal Reserve's Economic Database and pull data.

My motivation for putting this together was to develop a process more efficient that downloading .csv files and manually merging their contents in an Excel sheet.

In it's current state, it lives in a single .py file and requires manual coding from anyone that would seek to use it.
In the future, I plan to develop UI that can allow anyone to compile a dataset of their choosing.
